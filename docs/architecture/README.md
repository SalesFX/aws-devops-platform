# Arquitetura

Este diretório contém a documentação visual da arquitetura do projeto.

## Diagrama (Mermaid)

- Fonte: `docs/architecture/architecture.mmd`
- Imagem exportada (PNG): `docs/architecture/architecture.png`
- Como visualizar:
  - No GitHub, o Mermaid é renderizado automaticamente quando embutido em um Markdown.
  - No VS Code, use a extensão Mermaid/Markdown Preview.

### Exportar para PNG (opcional)

Opções comuns:

- GitHub: abra um Markdown com o Mermaid e use screenshot/print.
- CLI (se tiver `mmdc` instalado): exporte para PNG/SVG.
